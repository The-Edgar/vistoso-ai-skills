# Vistoso AI Skills

A collection of AI-powered creative tools for [Claude Code](https://claude.ai/claude-code), built by [Vistoso.ai](https://vistoso.ai).

## Available Plugins

| Plugin | Description |
|--------|-------------|
| [nano-banana-artist](./nano-banana-artist/) | Create any visuals with Nano Banana 2 (Gemini image generation) in seconds |

## Installation

Add this marketplace to Claude Code:

```bash
/plugin marketplace add The-Edgar/vistoso-ai-skills
```

Then install a plugin:

```bash
/plugin install nano-banana-artist@vistoso-ai-skills
```

## Requirements

- [Claude Code](https://claude.ai/claude-code)
- A Google Gemini API key (free tier available at https://aistudio.google.com/apikey)
